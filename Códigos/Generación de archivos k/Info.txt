Este código funciona para generar los archivos .k y sus respectivas carpetas, el código divide en subcarpetas para la ejecución en paralelo en un mismo computador. Es decir, el usuario puede indicar en cuantas subcarpetas desea dividir los archivos .k generados. Esto con el fin de paralelizar en un mismo computador el lanzamiento de las simulaciones para óptimizar la ejecución de las simulaciones.
